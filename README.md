# 萌娘TV弹幕网程序

### Install
```shell
composer create-project mengniang/mengniang --prefer-dist
```

or
```shell
            git clone https://github.com/donghaichen/mengniangtv.git
            cd mengniangtv
            composer install
            sudo php -S localhost:8888 -t public
            
```
